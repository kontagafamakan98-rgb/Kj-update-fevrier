# Assets Directory

This directory contains the app's assets including:

## Icons
- App icons for different screen densities
- Tab bar icons
- UI icons

## Images
- Splash screen backgrounds
- Placeholder images
- Country flags
- Category illustrations

## Fonts
- Custom fonts if any

## Sounds
- Notification sounds
- UI interaction sounds

## Usage
Import assets using relative paths:
```javascript
import logo from '../assets/images/logo.png';
```

For Expo managed workflow, assets in this directory are automatically bundled with the app.
