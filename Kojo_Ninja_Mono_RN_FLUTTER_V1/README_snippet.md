# Kojo - Installation rapide

## Prérequis
- Node.js 18+
- Android Studio
- Flutter SDK
- Git

## Installation
1. Cloner le repo
2. Exécuter `./scripts/setup_ninja_mono.sh`
3. Lancer `./bootstrap/boot_rn_flutter.sh`

## Structure
apps/
  rn-app/          # React Native
  flutter-app/     # Flutter
packages/
  shared/          # Code partagé
