## 🔄 CONVERSION DES FICHIERS

### Conversion en PDF

**Pitch Deck 150 pages :**
```bash
# Méthode 1 : Pandoc (Recommandé)
pandoc /app/KOJO_PITCH_150.md \
  -o /app/KOJO_PITCH_150.pdf \
  --toc \
  --toc-depth=2 \
  --number-sections

# Méthode 2 : En ligne
# Aller sur dillinger.io
# Importer KOJO_PITCH_150.md
# Export → PDF
```

**Présentation PowerPoint :**
```bash
# Méthode 1 : Marp (Recommandé)
npm install -g @marp-team/marp-cli

  -o /app/KOJO_PRESENTATION.pptx

# Utiliser template Pitch Deck
# Export → PowerPoint
```
